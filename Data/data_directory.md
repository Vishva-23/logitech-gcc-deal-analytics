# 📖 Data Dictionary — Logitech GCC Deal Dataset

## Source
Generated by Logitech's Global Customer Collaboration (GCC) team from their Salesforce CRM (SFDC). Each row represents one **product line item** within a B2B deal. A single deal (Deal ID) can contain multiple rows — one per product.

---

## Column Definitions

| # | Column Name | Type | Description |
|---|---|---|---|
| A | B2B Sales Region | Text | Sales territory: **EAD** (Europe/Africa/Middle East), **NAM** (North America), **GEM** (Emerging Markets), **GCN** (Greater China) |
| B | Company Name | Text | Customer company name (anonymised in dataset) |
| C | Fiscal Period | Text | Logitech fiscal period in format `YYYY-QN`. FY runs **April → March** (e.g. FY2026 = Apr 2025 – Mar 2026) |
| D | Created Date | Date | Date the deal was created in Salesforce |
| E | Close Date | Date | Planned or actual deal close date |
| F | Age | Integer | Number of days the deal has been in the system |
| G | Stage(SFDC) | Text | Deal stage: Closed Won / Pipeline / Commit / Upside / Need Assessment |
| H | Probability (%) | Numeric | Estimated win probability (1 = Closed Won) |
| I | Forecast Category | Text | Salesforce forecast bucket: Closed / Commit / Upside / Pipeline |
| J | Marketing Name | Text | Product marketing name (e.g. Rally Bar, MeetUp, TAP) |
| K | Product Description | Text | Full product description including variant/SKU details |
| L | Quantity | Integer | Number of units in this line item |
| M | Total Price (Local) | Numeric | Line item value in the local currency of the region |
| N | Total Price (converted USD) | Numeric | Line item value converted to USD — **use this for all cross-region analysis** |
| O | Product Number | Text | Unique product SKU identifier (e.g. 960-001073) |
| P | Product Line | Text | Internal product line code |
| Q | Life Cycle Status | Text | Product lifecycle: Active / LTS (Long Term Support) / EOM (End of Marketing) / Phase out / Obsolete |
| R | Last Modified Date | Date | Date the record was last updated in Salesforce |
| S | Deal ID | Text | Unique Salesforce opportunity identifier — use to aggregate multi-product deals |
| T | Product Group Name | Text | Product category (e.g. Video Collaboration Group, Webcams Group) |
| U | Company Segmentation | Text | Account tier: Global / Enterprise / SMB |
| V | Company Designation | Text | Strategic classification: Strategic / Named / Territory |
| W | Account Category | Text | Named or Territory |
| X | Account Record Type | Text | Record type — almost always "End Customer" in this dataset |
| **Y** | **Account Type** | **Text** | **DERIVED — Global Strategic Customer or Local Customer (see methodology)** |
| **Z** | **FY** | **Text** | **DERIVED — Fiscal year extracted from Fiscal Period** |
| **AA** | **FQ** | **Text** | **DERIVED — Fiscal quarter extracted from Fiscal Period** |
| AB | Is_EOL_Product | Boolean | BONUS DERIVED — True if Product Number matches the EOL SKUs reference list |

## Key Definitions

### Global Strategic Customer
```
Company Segmentation = "Global" AND Company Designation = "Strategic"
```
208 unique companies. 82,638 rows. Rationale: the dataset filter header states "Strategic Group Owner not equal to blank" — both conditions together identify formally assigned strategic accounts.

### Fiscal Year
Logitech FY runs **April 1 → March 31**. FY2026 = April 2025 to March 2026.  
⚠️ Never derive FY from Close Date using calendar year — this produces wrong results.

### Future Deal
```
Close Date > 2026-04-24 (analysis date)
```

### EOL Product
Product Number appears in the `EOL SKUs` reference sheet (14 discontinued SKUs).  
Do NOT use Life Cycle Status to determine EOL — it has 5 ambiguous values, none of which say "EOL" explicitly.

## Data Quality Notes

| Column | Null Rate | Notes |
|---|---|---|
| Life Cycle Status | 1.9% | Rows kept — revenue data still valid |
| Company Name | 0.3% | Rows kept — product/deal data still valid |
| Company Designation | ~small% | Rows with Global segmentation + null designation → classified as Local Customer |
| Product Group Name | ~0.9% | Excluded from product group analysis only |
